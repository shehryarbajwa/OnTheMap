# OnTheMap
An iOS app that allows you to login via Udacity API and share your location with fellow students

## Technologies Used:
CoreData <br>
MapKit <br>
Foundation <br>
UITableView <br>
UICollectionView <br>

## Frameworks Used:
UdacityAPI <br>
UdacityParseAPI <br>
